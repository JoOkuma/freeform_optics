# freeform_optics

Design a 3D gradient-index (GRIN) refractive optic via gradient descent using [drjit](https://github.com/mitsuba-renderer/drjit).

## Install

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
uv sync
```

## Run

```bash
uv run python main.py
```

Output images and loss curves are written to `output/`.
